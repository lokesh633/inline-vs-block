       Inline vs block-level elements
  Block
    1,Block element occupy full width
    2,Block always starts with new line
    3,Block level element does not allow another 
      block level element to sit next to the element
    4,Block level element have top and bottom margin
    Eg, div, h1, h2, h3, h4, h5, h6

  Inline
    1,Inline element only takes required width
    2,Inline doesn't starts with new line
    3,Inline element allow other inline element to sit 
      next to the element
    4,Inline element doesn't have top and bottom margin
    Eg, span, anchor, subscript and superscript
